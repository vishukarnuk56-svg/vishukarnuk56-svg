### 🏆 Key Project Portfolio

#### 🪐 **Bharata Yantra**
An intricate astronomical and geographical mapping instrument designed to synthesize micro-precision mechanical gear trains with coordinate frameworks. Integrates historical spatial paths, gear ratio kinematics, and structural CAD design.
* **Tech Stack:** SolidWorks, Blender, Advanced Kinematics, Precision Mechanical Modeling.

#### 📦 **MedInventory Pro**
A production-grade, standalone Python desktop framework engineered to manage intense local retail pharmacy operations. Handles high-integrity transactional databases, multithreaded calculation pipelines, and automated real-time billing workflows.
* **Tech Stack:** Python, SQLite/SQL, GUI Frameworks, Executable Compilers.

#### 🧭 **Autonomous SLAM Scanning Rover**
A self-contained ground vehicle engineered for automated environment scanning. Utilizes real-time LiDAR processing pipelines to construct precise digital spatial maps while navigating complex structural layouts.
* **Tech Stack:** Embedded C/C++, LiDAR Interfaces, Sensor Fusion, ROS, CAD Chassis Design.

#### ✒️ **Custom Mechanical Plotting System**
A precision X-Y coordinate plotting machine designed from scratch, translating raw software coordinate vectors into high-accuracy structural stepping movements.
* **Tech Stack:** G-Code Translation, Stepper Kinematics, Firmware Optimization, Hardware Integration.
