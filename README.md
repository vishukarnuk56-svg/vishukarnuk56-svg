# Hey, I'm Vishu 🛠️

I'm an Electronics and Communication Engineering undergrad (graduating 2028) with a background in Electronics and Telecommunication (EXTC). I build autonomous robotics, design complex multi-layer PCBs, and write custom software to control physical hardware. 

---

### 💻 What I Do

*   **Hardware Design:** Routing high-speed **8 to 18-layer PCBs**, signal integrity analysis, and hardware reverse engineering. 
*   **Robotics & Automation:** Building autonomous ground rovers, custom LiDAR-based SLAM scanners, and coordinate-plotting machines. I handle all my mechanical modeling and kinematics in **SolidWorks** and **Blender**.
*   **Software & DBs:** Writing multi-threaded desktop applications in Python with optimized local relational databases, built to handle heavy, real-time workflows.
*   **RF & Security:** Experimenting with digital signal detection, wireless network diagnostics, and packet injection using Kali Linux.
*   **Niche Interests:** Advanced cartography, processing LiDAR data, and exploring archaeoacoustics.

### 🚀 Projects I'm Building

#### 🪐 Bharata Yantra
A mechanical mapping instrument designed to sync physical gear ratios with astronomical and planetary paths. I handled the entire system architecture, gear train calculations, and precision structural layout in SolidWorks.

#### 📦 MedInventory Pro
A standalone Python desktop application built from scratch to automate high-volume retail workflows. It uses a custom GUI framework and a multithreaded database backend to handle instantaneous local billing logs and live inventory tracking without lagging.

#### 🧭 Autonomous SLAM Rover
A ground vehicle that processes real-time LiDAR point clouds to map out unknown rooms. Includes a custom-fabricated chassis modeled in CAD, sensor fusion firmware, and localized navigation tracking.
